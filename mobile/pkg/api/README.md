# Package

Api is a dart interface for application mobile in dairyfood.

## Installation

Import the package by inserting it on pubspec yaml of your project.

```yaml
dependencies:
  flutter:
    sdk: flutter
  common:
    path: ../
```
## Requirement

- firebase_auth
- simple_logger
