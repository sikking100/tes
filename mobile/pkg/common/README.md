# Package

Common is a global widget for application mobile in dairyfood.

## Installation

Import the package by inserting it on pubspec yaml of your project.

```yaml
dependencies:
  flutter:
    sdk: flutter
  common:
    path: ../
```
## Requirement

- google_fonts: ^3.0.1
- pin_code_fields: ^7.4.0
- package_info_plus: ^3.0.2
- share_plus: ^6.3.0
- intl: ^0.17.0
